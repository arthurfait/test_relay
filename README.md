# test_relay

## Description:
- This project is generally useless
- Aim of the project is to build relay socket communication to non-tcp channel.
    - Examples can be usb pipe and so on.

## Building:

Build using cmake.

#### make

```shell
mkdir build
cd build
cmake ..
make -j4
```

#### ninja

```shell
mkdir build
cd build
cmake -GNinja ..
ninja
```
